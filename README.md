## 🖥️ Projeto

Esse é um projeto de um jogo de biscoito da sorte usando DOM,onde cada vez aparece uma frase diferente.

## 🚀 Tecnologia

Esse projeto foi desenvolvido durante o curso Explorer da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- JavaScript
- Git e Github
